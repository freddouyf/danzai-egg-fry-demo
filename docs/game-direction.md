# 旦仔项目方向记录

## 新方向定义

《旦仔节奏厨房：今日营业》是一个“轻操作 + 强策略”的厨房肉鸽原型。TAP / HOLD / MASH / SWIPE 只负责让做菜有手感，核心上头点来自订单选择、厨具构筑、顾客风险、菜品收益和每局不同的营业路线。

## 为什么不是音游

节奏厨房实验版验证了动作输入的手感，但如果只追求 Good / Perfect / Combo，后续构筑空间会偏窄。今日营业把操作简化成订单执行，把主要决策放到“接什么单、拿什么厨具、承担多少混乱风险”上。

## 为什么不是纯闯关

纯闯关的目标通常是通过固定关卡。今日营业希望每局都有不同路线：这波接高风险订单还是稳定订单，这次拿金币流还是满意度流。玩家失败后应该会想“下局换一套构筑再试一次”。

## 今日营业核心循环

1. 进入今日营业。
2. 查看当前波次、金币、满意度、混乱值和已有厨具。
3. 每波从 3 个订单里选择 1 个。
4. 用轻量动作完成订单。
5. 根据订单标签和厨具获得金币、满意度和混乱变化。
6. 每波结束后三选一厨具。
7. 完成 3 波后结算本局。

## 玩家爽点

- 看订单标签，判断风险和收益。
- 选厨具后形成路线，例如煎蛋引擎、高价精品、早餐套餐、风险爆炒。
- 同一个订单在不同构筑下收益不同。
- 混乱值带来风险压力，满意度带来经营反馈。
- 一局短，失败原因容易复盘。

## v0.1 范围

- 3 波顾客。
- 每波 3 个订单，选择 1 个。
- 8 个策略型厨具。
- 每波结束后三选一强化。
- 结算页。
- 返回首页和重新开始。
- 不做后端、排行榜、复杂美术、复杂经济和正式存档。

## 旧玩法关系

当前保留「旦仔煎蛋挑战」和「节奏厨房实验版」。它们暂时作为可试玩原型并行存在，未来可以作为附加小游戏、训练关，或者在方向确定后删除。
